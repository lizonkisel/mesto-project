## Описание проекта

Проект представляет собой веб-страницу небольшой социальной сети для обмена фотографиями и впечатлениями.
На данном этапе сделана только вёрстка. Плнируется добавить функционал после изучения JS.


## Технологии проекта

* HTML
* CSS
* Адаптивная вёрстка
* Резиновая вёрстка
* Создание форм
